# 한글 표/그림 번호 재정렬 도구

한글(HWP) 문서에서 뒤죽박죽된 `[표 N]` / `<그림 N>` 번호를 문서 순서대로 1부터 자동 재정렬합니다.

---

## 사용법

### 실행 파일 (exe)

1. `hwp_renumber.exe` 실행
2. **찾아보기** 버튼으로 HWPX 파일 선택
   - 한글이 열려 있으면 파일 경로가 자동으로 입력됨
3. **▶ 실행** 클릭
4. 같은 폴더에 `원본파일명_renumbered.hwpx` 생성됨
5. 생성된 파일을 한글에서 열어 확인

### Python 스크립트

```bash
python hwp_renumber.py
```

---

## 요구 사항

| 항목 | 내용 |
|------|------|
| 운영체제 | Windows |
| 한글 | 한글 2024 (HWPX 형식으로 저장된 문서) |
| Python | 3.8 이상 (스크립트 실행 시) |
| 패키지 | `pywin32` (스크립트 실행 시) |

> exe 파일은 별도 설치 없이 실행 가능하나, **한글 2024가 설치된 PC**에서만 동작합니다.

---

## 동작 방식

- HWPX 파일은 내부적으로 ZIP + XML 구조
- HWP의 찾기/바꾸기 COM API 대신 XML을 직접 수정
  → 표 셀 안, 글상자 안 등 어디에 있어도 정상 동작
- HWP 자동번호 필드(`<hp:autoNum>`)가 있으면 자동 감지하여 번호 충돌 방지
- 원본 파일은 변경하지 않고 `_renumbered.hwpx`로 별도 저장

## 지원 패턴

| 패턴 | 예시 |
|------|------|

| 그림 | `<그림 176> 제목` → `<그림 1> 제목` |
