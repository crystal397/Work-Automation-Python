# 공기연장 간접비 보고서 — 자동화 및 검수 표

|---|---|---|
| 표지·제출문·요약문 | 계약 메타로 자동 치환 | 회사명·작성일 1회 확인 |
| 1.1.1 공사 개요표 | 계약서·변경계약서에서 자동 추출 | 출처 페이지 캡쳐와 표 1:1 비교 |

| 1.3 계약 현황·변경 다이어그램 | 회차별 일수·금액 자동 표 | 누적 일수 자동 검산, 결과 확인 |

| 2.1.2 공사 형태 | 장기계속/계속비/단년도 분기 자동 본문 | enum 값 1개 확인 |
| 2.1.3 입찰 방식 | 내역/총액입찰 분기 자동 본문 | enum 값 1개 확인 |
| 2.2 청구권 규정 | 법령 자동 인용 | 그대로 사용 |

| 3.2 청구의 근거 | 계약유형 분기 자동 인용 | 그대로 사용 |
| 4.1 공기연장 일수 산정 | 변경계약 일수 자동 산술 | 누계 자동 검증, 결과 확인 |
| 4.2 산정 방식 본문 | 표준 인용 + 요율 자동 인용 | 요율 4개(일관·이윤·산재·고용) 출처 셀 비교 |
| 4.3.2.1 대상인원 표 | 인원투입현황 엑셀에서 자동 추출 | 엑셀 행과 표 1:1 비교 |
| 4.3.2.2 ① 급여 (인원별) | 엑셀·명세서 PDF·OCR 자동 추출 | OCR 정확도 90% 미만만 집중 확인 |
| 4.3.2.2 ② 일수 (인원별) | 인사카드 OCR + 산정기간 자동 계산 | 출처 캡쳐와 비교 |
| 4.3.2.2 ③④⑤⑥⑦ 산식 | 정해진 공식으로 자동 계산 | 산식 1회 확인 (모든 행 동일) |
| 4.3.3.1 경비 11비목 | 영수증·전표 자동 합산 | 비목별 합계 결과 확인 |
| 4.3.3.2 산재·고용보험료 | 노무비 × 요율 자동 (원도급/하도급 분기) | 요율만 확인 |
| 4.3.4 일반관리비·이윤 | 산식 자동 ((소계)×4.5%, (소계+일관)×5.18%) | 요율만 확인 |
