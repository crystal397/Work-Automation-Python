# 📄 공기연장 간접비 보고서 자동화 (HWP Indirect Cost Reporter)

> 엑셀 산출 데이터를 한글(HWP) 보고서 템플릿에 자동으로 채워주는 Windows 자동화 도구입니다.

---

## ✨ 주요 기능

- 엑셀(`.xlsx`) 파일에서 데이터를 자동으로 읽어 한글(`.hwpx`) 템플릿에 입력
- 간접노무비 집계표, 퇴직금 명세, 전체 총괄표 등 **6개 표** 자동 완성
- 완성된 문서를 **HWP 및 PDF** 형식으로 동시 저장
- GUI 파일 탐색기로 간편하게 파일 선택

---

## 🗂️ 프로젝트 구조

```
hwp_reporter/
│
├── main.py                    # 메인 실행 파일 (파일 선택 → 읽기 → 입력 → 저장)
├── excel_reader.py            # 엑셀에서 데이터를 읽는 모듈
├── hwp_writer.py              # 한글 문서에 데이터를 입력하는 모듈
├── utils.py                   # 공통 헬퍼 함수 (포매터, 커서 이동 등)
├── requirements.txt           # 의존 패키지 목록
├── README.md
└── one_file/
    └── excel_to_hwp.py        # 단일 파일 버전 (모듈 분리 없이 전체 로직 통합)
```

## 🔧 설치 방법

### 1. 사전 요구사항

| 항목 | 버전 |
|------|------|
| OS | Windows 10 / 11 |
| Python | 3.10 이상 |
| 한글(HWP) | 2020 이상 (COM 자동화 지원 버전) |

### 2. 패키지 설치

```bash
pip install -r requirements.txt
```

---

## 🚀 사용 방법

```bash
python main.py
```

1. 파일 탐색기가 열리면 **엑셀 파일(`.xlsx`)** 을 선택합니다.
2. 이어서 **한글 템플릿 파일(`.hwpx`)** 을 선택합니다.
3. 자동으로 데이터가 입력되고, 실행 폴더에 결과 파일이 저장됩니다.

### 출력 파일

| 파일명 | 설명 |
|--------|------|
| `공기연장보고서_YYYYMMDD.hwp` | 한글 편집 가능 파일 |
| `공기연장보고서_YYYYMMDD.pdf` | PDF 배포용 파일 |

---

## 📋 엑셀 파일 구조 (필수 시트)

| 시트명 | 용도 |
|--------|------|
| `전체총괄표` | 공사 기간, 비율, 총괄 금액 |
| `1-1. 간접노무비 집계표` | 간접노무비 항목별 명세 |
| `1-3. 퇴직금` | 직원별 급여 및 퇴직금 |

---

## 📝 한글 템플릿 누름틀 목록

템플릿 `.hwpx` 파일에는 아래 누름틀(필드)이 정의되어 있어야 합니다.

| 누름틀 이름 | 내용 |
|-------------|------|
| `start_day` | 공사 시작일 |
| `end_day` | 공사 종료일 |
| `duration` | 공기 연장 일수 |
| `rate_san` / `rate_go` / `rate_il` | 산재·고용·일반관리비율 |
| `table_start` | 첫 번째 표 시작 셀 |
| `second_table_start` | 두 번째 표 시작 셀 |
| `third_table_start` | 세 번째 표 시작 셀 |
| `third_total_salary` / `third_total_severance` / `third_total_sum` | 퇴직금 합계 |
| `fourth_table_start` | 네 번째 표 시작 셀 |
| `fifth_table_start` | 다섯 번째 표 시작 셀 |
| `sixth_table_start` | 여섯 번째 표 시작 셀 |

## ⚠️ 주의사항

- 반드시 **Windows 환경**에서 실행해야 합니다 (한글 COM 자동화 사용).
- 실행 전 한글 프로그램이 설치되어 있어야 합니다.
- 엑셀 파일은 수식 결과값이 저장된 상태(저장 후 닫힌 파일)여야 정확히 읽힙니다.
