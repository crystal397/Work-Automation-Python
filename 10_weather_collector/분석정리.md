# 10_weather_collector — 분석 정리

## 최종 목표

**건설현장 기상데이터 수집 & 작업불가일 산정 프로그램** — 비개발자(현장·계약 담당자)가
설치 없이 .exe를 더블클릭하여 다음 두 가지를 처리할 수 있는 독립 실행형 도구.

1. 기상청 ASOS 공식 관측 데이터를 지정 현장·기간으로 자동 수집 → SQLite 저장
2. 공종별(토공사·RC공사·타워크레인 등) 작업불가일을 판정 기준에 따라 산정 → 엑셀 보고서 자동 출력

**활용 목적**: 공기 지연 불가항력 입증 자료, 계약·클레임 대응, 간접비·장비 대기비 산정 근거

## 수정 이력 (2026-04-16)

### 1. 결측값 오판정 버그 수정 — `kma_client.py`

**문제**: API 응답에서 관측값이 없을 때 `0.0`으로 치환하여 오판정 발생

- `sunshine_hours = 0.0` → `0.0 < 2` → `is_no_sunshine = True` (관측 없는 날이 일조 부족으로 산정)
- `ground_temp = 0.0` → `0.0 <= 0` → `is_freeze_day = True` (관측 없는 날이 지면 동결로 산정)

**해결**: 결측값은 `None` 반환, `cmp()` 헬퍼에서 `None`이면 무조건 `False`

```python
def f(key):
    v = raw.get(key, "")
    if v is None or v == "":
        return None   # ← 0.0 대신 None
    return float(v)

def cmp(val, op, threshold):
    if val is None:
        return False  # ← 결측 = 미해당
    ...
```

- 엑셀 요약 시트: 기본값과 다른 기준값 사용 시 `우천 (기준 ≥5mm)` 형태로 병기.

```python
def daily_update():

    run_bulk_collection(start_date=yesterday, end_date=yesterday)
```

```python
lat, lon = None, None

    if lon_str: lon = float(lon_str)
except ValueError:
    print("  ※ 좌표값이 올바르지 않아 좌표 없이 진행합니다.")
    lat, lon = None, None
```

## 작업불가일 판정 기준 (12개 플래그)

표의 기준값은 기본값이며, GUI/CLI에서 공종별로 자유롭게 변경 가능.

| 플래그 | 기본 기준 | 관련 공종 |
|---|---|---|
| `is_rain_day` | 일강수량 ≥ 10mm | 토공사, RC, 도장, 강구조, 포장 |
| `rain_yn` | 소량 포함 강수 발생 (iscs) | 도장·방수 |
| `is_wind_day` | 최대풍속 ≥ 14m/s | RC, 타워크레인, 강구조 |
| `is_wind_crane` | 순간최대풍속 ≥ 10m/s | 타워크레인 |
| `is_snow_day` | 최대적설 ≥ 1cm | 토공사, 포장 |
| `snow_yn` | 눈·진눈깨비 발생 (iscs) | (직접 선택) |
| `is_heat_day` | 최고기온 ≥ 35℃ | RC, 강구조, 포장 |
| `is_cold_day` | 최저기온 ≤ -10℃ | 토공사, RC, 도장, 강구조, 포장 |
| `is_no_sunshine` | 일조시간 < 2hr | 도장·방수 |
| `is_freeze_day` | 지면온도 ≤ 0℃ | 토공사, RC, 도장, 포장 |
| `is_high_evap_day` | 증발량 ≥ 10mm | (직접 선택) |
| `fog_yn` | 안개 발생 (iscs) | 타워크레인 |

> `rain_yn` / `snow_yn` / `fog_yn` 은 iscs 날씨코드 기반 범주형 플래그로 수치 기준값 없음.
