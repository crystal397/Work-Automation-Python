# 27. 사진대지 엑셀 자동 삽입

사진 폴더의 이미지를 엑셀 사진대지 템플릿의 슬롯에 자동으로 삽입하는 CLI 도구.

---

## 사용 방법

```bash
pip install openpyxl pillow

# 기본 (출력 파일명 자동 생성: 원본파일명_완성.xlsx)
python insert_photos.py 사진대지.xlsx ./photos/

# 출력 파일명 지정
python insert_photos.py 사진대지.xlsx ./photos/ 결과물.xlsx
```

---

## 동작 방식

1. **슬롯 자동 탐지** — 엑셀 시트에서 일정 크기 이상의 병합 셀 영역을 사진 슬롯으로 인식
   - 기본 기준: 병합 행 수 ≥ 5, 병합 열 수 ≥ 4 (`MIN_SLOT_ROWS`, `MIN_SLOT_COLS`)
2. **사진 정렬** — 사진 폴더의 이미지 파일을 파일명 오름차순으로 정렬
3. **삽입** — 슬롯 순서대로 (위→아래) 사진을 하나씩 배치
   - 비율 유지 리사이즈 후 슬롯 크기의 **흰 캔버스 중앙에 붙여넣기** → 캔버스를 슬롯 전체에 고정
   - 슬롯 경계를 절대 벗어나지 않음, 엑셀 앵커 해석 방식에 무관하게 항상 중앙 정렬
   - EXIF 회전 정보 자동 반영 (스마트폰 세로 사진)
4. **저장** — 원본은 그대로 두고 별도 파일로 저장

---

## 주요 설정값

스크립트 상단에서 직접 수정 가능.

| 변수 | 기본값 | 설명 |
|------|--------|------|
| `MIN_SLOT_ROWS` | `5` | 슬롯 인식 최소 병합 행 수 |
| `MIN_SLOT_COLS` | `4` | 슬롯 인식 최소 병합 열 수 |
| `SLOT_PADDING_PX` | `4` | 슬롯 테두리-사진 간 여백 (px) |
| `JPEG_QUALITY` | `85` | 삽입 이미지 JPEG 품질 (1~95) |
| `IMAGE_EXTS` | `.jpg .jpeg .png .gif .bmp` | 인식할 이미지 확장자 |

---

## 중앙 정렬 방식

슬롯 크기의 흰 배경 캔버스를 생성하고 리사이즈된 사진을 그 중앙에 붙인 뒤,
캔버스 전체를 슬롯 영역에 고정(`TwoCellAnchor`)합니다.

엑셀의 앵커 오프셋 계산 오차에 관계없이 항상 정확한 중앙 배치가 보장됩니다.
