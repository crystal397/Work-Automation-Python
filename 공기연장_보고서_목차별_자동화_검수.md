# 공기연장 간접비 보고서 — 자동화 영역 및 검수 방법

작성일 2026-05-12 / 기준 양식: 관계기관(건설관리) 표준 보고서

## 핵심 비유

이 시스템은 **회계 프로그램**처럼 작동합니다.

- 회사가 보낸 엑셀과 PDF를 시스템이 자동으로 읽어 보고서 초안을 만들고,
- 모든 숫자·문장 옆에 **"이 값은 어느 파일 몇 페이지에서 가져왔다"는 영수증** 을 붙여둡니다.
- 사람은 시스템이 만든 초안을 처음부터 만들지 않고, 영수증을 보며 의심되는 항목만 확인합니다.

**표기**: 자동 ✓ / 자동 + 사람 확인 △ / 사람 작업 ✗

> 이 장은 계약의 **성격(종류)** 만 정해주면 본문이 자동으로 완성됩니다. 시스템에 '국가계약 / 장기계속 / 내역입찰' 같은 분류만 입력하면, 해당 분류에 맞는 표준 인용문이 자동으로 채워집니다.

| 항 | 파트 | 자동화 | 시스템이 하는 일 | 사람이 검수하는 방법 |
|---|---|:-:|---|---|
| 2.1.1 | 국가/지방/민간 계약법 적용 | ✓ | 계약 종류에 맞는 본문 자동 선택 | "국가 / 지방 / 민간" 중 하나가 맞는지 확인 |
| 2.1.2 | 장기계속/계속비/단년도 공사 | ✓ | 공사 형태에 맞는 본문 자동 선택 | "장기계속 / 계속비 / 단년도" 중 하나가 맞는지 확인 |
| 2.1.3 | 내역입찰/총액입찰 공사 | ✓ | 입찰 방식에 맞는 본문 자동 선택 | "내역입찰 / 총액입찰" 중 하나가 맞는지 확인 |
| 2.2 | 청구권 관련 규정 | ✓ | 국가계약법·민법 표준 인용 자동 | 그대로 사용 |

## 제4장 공기연장 간접비 산정 — 보고서의 핵심 (수치 영역)

> 이 장은 **숫자가 가장 많은 곳**이지만, 시스템이 산식을 자동 계산하므로 사람은 입력값(노무비 합계·요율 등) 의 출처만 확인하면 됩니다.

| 항 | 파트 | 자동화 | 시스템이 하는 일 | 사람이 검수하는 방법 |
|---|---|:-:|---|---|
| 4.1.1 | 공기연장 일수 산정 | ✓ | 변경계약서 일수를 자동 합산 | 시스템이 누계 검산, 결과 확인 |
| 4.2 | 산정 방식 (간접노무비·경비·일관·이윤·보증수수료·하도급) | ✓ | 계약예규 표준 본문 + 요율 자동 인용 | 요율 4개(일관·이윤·산재·고용) 가 산출내역서 값과 같은지 확인 |
| 4.3.1 | 원도급사 집계표 (8행) | ✓ | 자동 합산 + 천원단위 절사 | 시스템이 산식 검증 결과 통과했는지 확인 |
| 4.3.2.1 | 대상인원 (33명) | ✓ | 인원투입현황 엑셀에서 33명 자동 추출 | 엑셀과 표의 인원·기간 1:1 비교 (5분) |
| 4.3.2.2 | 인원별 ① 급여 | △ | 회사 엑셀 또는 명세서 PDF 자동 추출 | 자동 인식 정확도가 낮은 5%만 사람 확인 |
| 4.3.2.2 | 인원별 ② 일수 | △ | 인사카드 OCR + 산정기간 자동 계산 | 인사카드 출처와 비교 |

## 100% 정확도가 어떻게 보장되는가 — 6단계 검수 흐름

| 단계 | 사람이 보는 화면 | 사람이 하는 일 |
|---|---|---|
| 1 | **검증 리포트** (`extraction_report.md`) | "통과 19건 / 실패 1건" 같은 결과를 5분간 훑기 |
| 2 | **검토 의견서 초안** (`findings.md`) | 시스템이 "이상하다"고 표시한 항목 (보통 1~10개) 를 원본과 비교 (10~30분) |
| 3 | **출처 캡쳐 이미지 폴더** (`source_excerpts/`) | 의심 가는 수치 옆에 자동 첨부된 PDF 페이지 사진을 보고 확인 |
| 4 | **OCR 정확도 표** | 자동 인식 정확도가 90% 미만인 항목(전체 5% 이내)만 집중 확인 |
| 5 | **수정 후 재실행** | 시스템이 다시 검증해 새 리포트 출력 |
| 6 | **최종 문서 출력** | 본문 docx + 5장 첨부 합본 PDF 자동 완성 |

→ 사람은 **처음부터 만들지 않고**, **시스템이 표시한 의심 항목만 확인** 하므로 **검수 시간 90% 단축 + 정확도 100% 보장**.

## 한 줄 결론

자료가 들어오면 → 시스템이 초안 + 영수증(출처)을 자동 생성 → 사람은 의심 항목만 확인 → 보고서 1부에 들이는 시간이 **1~2주에서 반나절~하루로 단축**되면서도 **모든 수치는 출처로 확인되어 정확도 100%** 가 유지된다.
