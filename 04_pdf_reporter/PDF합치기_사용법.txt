================================================================================
  PDF 합치기 + 갑지 생성 도구 v3 사용법
================================================================================

1. 프로그램 개요
--------------------------------------------------------------------------------
  폴더 구조를 기반으로 PDF와 Excel 파일을 하나의 PDF로 자동 합치는 도구입니다.
  각 파일/폴더 앞에 갑지(표지) 페이지를 자동 생성하고, PDF 북마크(목차)도
  함께 만들어줍니다.

  주요 기능:
    - 폴더 계층 구조에 따라 자동 번호 부여 (예: 1. → 1.1. → 1.1.1.)
    - 각 항목 앞에 갑지(표지) 페이지 자동 삽입
    - 초기 갑지(맨 앞에 삽입할 표지) 자유롭게 추가/삭제/순서 변경
    - 갑지 번호 시작 번호 지정 가능
    - Excel 파일(.xlsx, .xls, .xlsm) 자동 PDF 변환 후 합치기
    - PDF 북마크(목차) 자동 생성
    - GUI 모드 및 CLI(명령줄) 모드 모두 지원


2. 폴더 구조 준비 (중요)
--------------------------------------------------------------------------------
  프로그램은 폴더 이름과 파일 이름의 앞쪽 번호를 기준으로 자동 정렬하고
  계층 번호를 부여합니다.

  권장 폴더 구조 예시:

      루트폴더/
      ├── 1. 공사개요/
      │   ├── 1.1 공사 요약.pdf
      │   └── 1.2 현장 사진.pdf
      ├── 2. 공사비 내역/
      │   ├── 2.1 직접공사비/
      │   │   ├── 1. 자재비.xlsx
      │   │   └── 2. 노무비.pdf
      │   └── 2.2 간접공사비.pdf
      └── 3. 첨부서류/
          └── 도면목록.pdf          ← 번호 없는 파일은 번호 없이 처리

  결과 PDF 구조:

      [갑지] 1. 공사개요
        1.1 공사 요약.pdf
        1.2 현장 사진.pdf
      [갑지] 2. 공사비 내역
        [갑지] 2.1. 직접공사비
          2.1.1. 자재비 (Excel→PDF 변환)
          2.1.2. 노무비.pdf
        2.2. 간접공사비.pdf
      [갑지] 3. 첨부서류
        도면목록.pdf


3. 실행 방법 A: GUI 모드 (기본)
--------------------------------------------------------------------------------

  프로그램 창이 열리면 아래 순서로 진행합니다.

  [STEP 1] 루트 폴더 선택
    - "찾아보기" 버튼을 눌러 합칠 파일들이 있는 최상위 폴더를 선택합니다.
    - 출력 PDF 경로가 자동으로 설정됩니다. (폴더명_merged.pdf)

  [STEP 2] 출력 PDF 경로 확인/변경
    - 필요하면 "저장 위치" 버튼을 눌러 출력 파일 위치를 변경합니다.

  [STEP 3] 시작 번호 설정
    - 기본값은 1입니다.
    - 예: 3으로 설정하면 첫 번째 항목이 "3. xxx"부터 시작합니다.
    - 이전 문서에 이어서 번호를 매길 때 유용합니다.

  [STEP 4] 초기 갑지 설정
    - 합쳐진 PDF의 맨 앞에 삽입할 갑지(표지)를 설정합니다.
    - 기본으로 "# 첨부자료", "추가공사비 집계표" 2개가 추가되어 있습니다.
    - "＋ 갑지 추가" 버튼으로 새 갑지를 추가할 수 있습니다.
    - 각 갑지의 ▲/▼ 버튼으로 순서를 변경하고, ✕ 버튼으로 삭제합니다.
    - 필요 없으면 모두 삭제해도 됩니다.

  [STEP 5] 합치기 시작
    - "▶ 합치기 시작" 버튼을 누르면 처리가 시작됩니다.
    - 하단 로그 창에서 진행 상황을 실시간으로 확인할 수 있습니다.
    - 완료 시 총 페이지 수, 갑지 수, PDF/Excel 파일 수가 표시됩니다.


4. 파일 처리 규칙
--------------------------------------------------------------------------------
  (1) 지원 파일 형식
      - PDF (.pdf): 그대로 합치기
      - Excel (.xlsx, .xls, .xlsm): PDF로 변환 후 합치기

  (2) 번호 부여 규칙
      - 파일/폴더 이름이 숫자로 시작하면 (예: "1. 공사개요") 자동 계층 번호 부여
      - 숫자로 시작하지 않으면 (예: "도면목록.pdf") 번호 없이 이름만 표시

  (3) 정렬 기준
      - 파일/폴더 이름 앞쪽의 숫자를 기준으로 자동 정렬
      - 예: "1.1", "1.2", "1.10" 순으로 올바르게 정렬

  (4) 자동 무시 항목
      - 숨김 파일 (이름이 .으로 시작)
      - 임시 파일 (이름이 ~$로 시작)
      - PDF, Excel 외 다른 형식의 파일


5. 결과물
--------------------------------------------------------------------------------
  - 하나의 PDF 파일로 합쳐지며, PDF 뷰어에서 북마크(목차)를 통해
    원하는 항목으로 바로 이동할 수 있습니다.
  - 각 항목 앞에 갑지(표지) 페이지가 삽입되어 구분이 용이합니다.


6. 참고 사항
--------------------------------------------------------------------------------
  - Excel 변환 기능은 Windows + Microsoft Excel 환경에서만 작동합니다.
  - 처리 중에는 Excel이 백그라운드에서 실행되므로, 다른 Excel 파일을
    동시에 작업하지 않는 것을 권장합니다.
  - 대용량 폴더 처리 시 시간이 다소 소요될 수 있습니다.
