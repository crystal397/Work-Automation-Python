# PDF 자동 합치기 + 갑지 생성 도구

폴더 구조를 순회하며 **갑지(표지)를 자동 생성**하고, PDF와 Excel 파일을 **하나의 PDF로 합쳐주는 도구**입니다.

---

## 📋 주요 기능

| 기능 | 설명 |
|---|---|
| 갑지 자동 생성 | 폴더/파일명을 읽어 계층 번호가 붙은 갑지를 자동 삽입 |
| 계층 번호 자동 부여 | 1단계 → `1.`, 2단계 → `1.1.`, 3단계 → `1.1.1.` |
| Excel → PDF 변환 | Microsoft Excel로 전체 시트를 PDF로 자동 변환 |
| 파일 정렬 | 파일명 앞 숫자 기준으로 자동 정렬 (1., 2., 1.1., ...) |
| 초기 갑지 커스텀 | GUI에서 맨 앞 갑지를 자유롭게 추가/삭제/순서변경 (v3) |
| 시작 번호 지정 | 첫 번째 항목의 번호를 GUI에서 직접 지정 (예: 4부터 → 4., 5., 6. ...) (v3) |
| 북마크 자동 생성 | PDF 합치기 시 목차(아웃라인) 자동 포함 (v3) |
| 번호 없는 항목 지원 | 파일명에 번호가 없으면 번호 없이 이름만 갑지에 표기 (v3) |

## 📁 갑지 생성 규칙

```
산출물/                              ← 루트 폴더 (갑지 없음)
├── 1. 업무지시서/                   ← 갑지: "1. 업무지시서"
│   ├── 1.1. 수직구/                 ← 갑지: "1.1. 수직구"
│   │   ├── 1. 내역서.pdf            ← 갑지: "1.1.1. 내역서" + PDF 본문
│   │   └── 2. 산출서.xlsx           ← 갑지: "1.1.2. 산출서" + Excel→PDF
│   └── 1.2. 터널/                   ← 갑지: "1.2. 터널"
│       └── 1. 도면.pdf              ← 갑지: "1.2.1. 도면" + PDF 본문
└── 2. 공기만회/                     ← 갑지: "2. 공기만회"
    └── 1. 계획서.pdf                ← 갑지: "2.1. 계획서" + PDF 본문
```

**최종 PDF 순서:**
```
[# 첨부자료] → [추가공사비 집계표] → [1. 업무지시서] → [1.1. 수직구]
→ [1.1.1. 내역서] → 내역서.pdf → [1.1.2. 산출서] → 산출서.xlsx(PDF변환)
→ [1.2. 터널] → [1.2.1. 도면] → 도면.pdf → ...
```

## 🚀 사용 방법

### 방법 1: EXE 파일 사용 (권장)

빌드 후 생성된 `dist/pdf_merger.exe`를 실행하면 GUI가 열립니다.

```
pdf_merger.exe            ← GUI 실행
pdf_merger.exe C:\산출물  ← 폴더 지정하여 GUI 실행
```

> GUI에서 루트 폴더, 출력 경로, 초기 갑지, 시작 번호를 설정한 뒤 **합치기 시작**을 클릭합니다.

---

### 방법 2: Python으로 직접 실행

```bash
python pdf_merger_v3.py
```

---

## 🔨 EXE 빌드 방법

`build.ps1`을 **우클릭 → PowerShell로 실행**합니다.

```
우클릭 → PowerShell로 실행
```

빌드가 완료되면 `dist/pdf_merger.exe` 파일이 생성됩니다. (`pdf_merger_v3.py` 기준으로 빌드)

> **최초 1회만** 빌드하면 됩니다. 이후에는 exe 파일만 배포하면 됩니다.

## ⚠️ 주의사항

- **Excel 파일 변환 시** Microsoft Excel이 백그라운드에서 잠깐 실행됩니다. 변환 중에는 Excel을 직접 사용하지 마세요.
- **파일 정렬**은 파일명 맨 앞의 숫자 기준입니다. 숫자가 없는 파일은 숫자 있는 파일 뒤에 배치됩니다.
- **한글 폰트**가 없으면 갑지 텍스트가 깨질 수 있습니다. 한컴오피스 또는 나눔폰트를 설치해 주세요.
- **파일명에 특수문자**(`<>:"/\|?*`)가 포함된 경우 갑지 생성이 실패할 수 있으므로 사전에 제거해 주세요.
- `~$`로 시작하는 임시 파일(Excel 열려있는 상태)은 자동으로 건너뜁니다.

## 🐛 오류 해결

| 증상 | 원인 | 해결 방법 |
|---|---|---|
| 한글이 깨져서 나옴 | 한컴바탕 폰트 없음 | 한컴오피스 또는 나눔폰트 설치 |
| Excel 변환 실패 | Microsoft Excel 미설치 | Excel 설치 후 재시도 |
| `pywin32` 오류 | 패키지 미설치 | `pip install pywin32` 실행 |
| EXE 실행 시 바이러스 경고 | 백신 오탐 | 백신 예외 처리 또는 직접 빌드 |
| 처리된 파일이 없습니다 | 폴더 내 PDF/Excel 없음 | 폴더 경로 및 파일 확인 |
