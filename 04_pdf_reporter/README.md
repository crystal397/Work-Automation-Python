# 📊 Excel → PDF 변환기

Excel 파일에서 **원하는 시트만 선택**해 PDF로 변환하는 Windows GUI 프로그램입니다.  
자주 제외하는 시트는 **규칙으로 미리 등록**해두면, 파일을 열 때 자동으로 체크 해제됩니다.

---

## 📁 파일 구성

```
excel_to_pdf.py          # 메인 소스 코드
excel_pdf_config.json    # 제외 규칙 저장 파일 (최초 실행 시 자동 생성)
build.bat                # .exe 빌드 스크립트
README.md                # 이 문서
```

---

## ⚙️ 시스템 요구사항

| 항목 | 내용 |
|------|------|
| 운영체제 | **Windows** 전용 |
| Microsoft Excel | **설치 필수** (Excel COM을 통해 PDF 출력) |
| Python | 3.9 이상 |
| 필수 패키지 | `pywin32`, `openpyxl` |

> ⚠️ Microsoft Excel이 설치되어 있지 않으면 변환이 동작하지 않습니다.

---

## 🚀 설치 및 실행

### 방법 1 — Python으로 직접 실행

```bash
pip install pywin32 openpyxl
python excel_to_pdf.py
```

### 방법 2 — .exe 파일로 빌드 후 실행

`build.bat`을 더블클릭하면 패키지 설치부터 빌드까지 자동으로 진행됩니다.

```
build.bat 더블클릭
→ dist/Excel_PDF_변환기.exe 생성
```

빌드 내부 순서:

```
1. pip install pyinstaller pywin32 openpyxl
2. pyinstaller --onefile --windowed --name "Excel_PDF_변환기" excel_to_pdf.py
3. dist/ 폴더에 단일 .exe 파일 생성
```

> 💡 `.exe` 파일을 다른 PC에 배포할 때는 해당 PC에도 **Microsoft Excel**이 설치되어 있어야 합니다.

## 🖥️ 화면 구성 및 사용 방법

### 메인 화면

```
┌─────────────────────────────────────────────┐
│  📊 Excel → PDF 변환기     [⚙ 제외 규칙 설정] │  ← 헤더
├─────────────────────────────────────────────┤
│  ① Excel 파일 선택                           │
│  [ 파일 경로 표시                   ] [찾아보기] │
├─────────────────────────────────────────────┤
│  ② 변환할 시트 선택                          │
│  ┌─────────────────────────────────────────┐ │
│  │ ☑ 시트1                                │ │  ← 체크박스 목록
│  │ ☑ 시트2                                │ │    (스크롤 가능)
│  │ ☐ 임시_백업   [자동 제외]              │ │  ← 규칙 매칭 시 배지 표시
│  └─────────────────────────────────────────┘ │
│  [전체 선택] [전체 해제] [↺ 규칙 재적용]      │
│  🚫 자동 제외 규칙: "임시"(포함)              │  ← 활성 규칙 힌트
├─────────────────────────────────────────────┤
│  ③ PDF 저장 위치                             │
│  [ 저장 경로 표시                   ] [찾아보기] │
├─────────────────────────────────────────────┤
│  ████████████░░░░  변환 준비 중...            │  ← 진행 바
│            [ PDF로 변환하기 ]                 │
└─────────────────────────────────────────────┘
```

### 사용 순서

1. **① Excel 파일 선택** — `찾아보기` 버튼으로 `.xlsx` / `.xlsm` / `.xls` 파일 선택  
   → 시트 목록이 자동으로 로드되고, 등록된 규칙에 해당하는 시트는 자동 체크 해제됩니다.

2. **② 시트 선택** — 체크박스로 PDF에 포함할 시트를 직접 선택/해제

3. **③ 저장 위치 지정** — 기본값은 `원본파일명_converted.pdf` (같은 폴더)

4. **PDF로 변환하기** 버튼 클릭 → 완료 후 PDF를 바로 열지 선택 가능

## 🚫 제외 규칙 설정

헤더 오른쪽의 **⚙ 제외 규칙 설정** 버튼을 클릭하면 규칙 관리 창이 열립니다.

### 규칙 타입

| 타입 | 동작 | 예시 |
|------|------|------|
| `keyword` | 시트 이름에 해당 문자열이 **포함**되면 제외 | `임시` → "임시데이터", "임시_백업" 모두 제외 |
| `exact` | 시트 이름이 **정확히 일치**하면 제외 | `Sheet1` → "Sheet1"만 제외, "Sheet11"은 미해당 |

> 대소문자를 구분하지 않습니다. (`keyword` 타입)

### 규칙 관리

- **추가** — 타입 선택 후 값 입력 → `+ 추가` 클릭
- **비활성화** — 각 규칙 왼쪽 체크박스로 규칙을 삭제하지 않고 임시로 끄거나 켤 수 있음
- **삭제** — 각 규칙 오른쪽 `✕` 버튼 클릭
- **저장** — `저장` 버튼 클릭 시 `excel_pdf_config.json`에 저장되고, 현재 로드된 시트 목록에 즉시 재적용

### 규칙 재적용

시트 목록을 수동으로 수정한 뒤 다시 규칙 기준으로 초기화하고 싶을 때 **↺ 규칙 재적용** 버튼을 사용합니다.

---

## 🗂️ 설정 파일 (excel_pdf_config.json)

규칙은 실행 파일과 같은 폴더에 JSON 형식으로 저장됩니다.

```json
{
  "rules": [
    {
      "type": "keyword",
      "value": "임시",
      "enabled": true
    },
    {
      "type": "exact",
      "value": "Sheet1",
      "enabled": true
    },
    {
      "type": "keyword",
      "value": "백업",
      "enabled": false
    }
  ]
}
```

| 필드 | 설명 |
|------|------|
| `type` | `"keyword"` 또는 `"exact"` |
| `value` | 규칙 값 (시트 이름 또는 포함 문자열) |
| `enabled` | `true` = 활성, `false` = 비활성 |

## 🔧 변환 동작 방식

내부적으로 **Microsoft Excel COM 자동화**를 사용합니다.

```
1. Excel을 백그라운드에서 실행 (화면에 표시 안 됨)
2. 선택되지 않은 시트를 임시로 숨김 (xlSheetHidden)
   - 이미 완전히 숨겨진 시트(xlSheetVeryHidden)는 건드리지 않음
3. 워크북 전체 ExportAsFixedFormat() 호출
   → 보이는 시트만 PDF로 출력됨
4. 숨긴 시트를 원래 상태로 복원
5. 파일 저장 없이 닫기
```

> 원본 Excel 파일은 **읽기 전용(ReadOnly=True)**으로 열리므로 내용이 변경되지 않습니다.

## ❗ 주의사항

- 변환 중에는 Excel이 백그라운드에서 실행됩니다. 변환이 끝날 때까지 해당 파일을 직접 열지 마세요.
- `.xlsm` 파일(매크로 포함)은 Excel 보안 설정에 따라 경고가 발생할 수 있습니다.
- 인쇄 영역(Print Area)이 설정된 시트는 해당 영역만 PDF에 포함됩니다.  
  전체 내용을 출력하려면 Excel에서 인쇄 영역을 해제하세요.
