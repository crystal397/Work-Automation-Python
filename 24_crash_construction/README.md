# 24. 돌관공사비 노무비 출력일보 자동 작성

소스 Excel 파일(업체별 공수 내역)을 읽어 **돌관공사비 산출근거 — 노무비 출력일보** 시트를 자동으로 생성하는 도구.

---

```

├── mandays_report_automation_v7.py       ← 페이지 분할·AB/AC 단가·총액 추출 (v7)
├── mandays_report_automation_v8.py       ← 최신: 공휴일 자동 처리·연도별 분할 저장 (v8)
├── sample_260415_케이씨산업 돌관공사비 산출근거.xlsx   ← 기본 템플릿
├── source/                               ← 소스 Excel 파일 (파일명에 연/월 포함)

    ...
```

## 사용 방법

```bash
pip install openpyxl holidays
python mandays_report_automation_v8.py
```

경로는 스크립트 상단 **사용자 설정** 섹션에서 수정:

```python
TEMPLATE_PATH = r"...\sample_돌관공사비 산출근거.xlsx"
SOURCE_DIR    = r"...\source"
OUTPUT_PATH   = r"...\돌관공사비_산출근거_자동생성_v8.xlsx"
```

실행 후 Excel에서 **Ctrl+Alt+F9** (전체 재계산) 실행.

## 소스 파일 구조

- 탭 이름: `01.가시설`, `02.형틀목공` 등 `숫자.공종명` 형식
- B열 `작업자명` 헤더 아래 행: 작업자 이름
- C열: 일자(1~31), D열: 공수, F·H열: 카테고리 키워드 (`본선`/`복합`/`삼성` → 없으면 `기타`)
- B열 `일 기준` 아래 값: 노무비 단가 (AB열)
- 블록 하단 E열 마지막 숫자: 노무비 총액 (AC열)

---

## 출력 구조

- **카테고리 4종**: 본선 / 복합 / 삼성 / 기타
- **페이지 분할**: 한 페이지 최대 21명 — 초과 시 헤더를 재출력하며 다음 페이지로 분할
- **합계행**: 마지막 페이지에만 삽입, E열에 `N명` 자동 기입, 수식 범위는 전체 페이지 기준
- **날짜 헤더 색상**:
  - 토요일 → 파란 글씨
  - 일요일 · 공휴일 → 빨간 글씨 + `휴` 표시
- **연도별 분할**: `노무비 출력일보` 시트만 포함한 `_YYYY.xlsx` 별도 저장
