# 물가변동 비목군 자동분류 프로그램

산출근거(산근) 엑셀 파일의 각 행에 비목군 기호를 자동으로 분류하여 A열에 기입하는 프로그램입니다.

**법적 근거**: 국가계약법 계약예규 제68조, 조달청 물가변동 질의응답집(2025.10)

---

## 비목군이란?

물가변동(계약금액 조정) 시 각 품목의 가격변동률을 어떤 물가지수로 적용할지 결정하는 분류 기호입니다. 계약예규 제68조에서 정한 8개 비목군으로 구분합니다.

| 기호 | 명칭 | 해당 품목 예시 |
|------|------|----------------|
| A | 노무비 | 노동자 임금, 인건비 |
| B | 기계경비 | 크레인, 굴삭기 등 건설기계 가동비 |
| C | 광산품 | 잡석, 골재, 모래, 화강암 등 |
| D | 공산품 | 철근, 시멘트, 페인트 등 공장 생산품 |
| E | 전력·수도·가스·폐기물 | 전기료, 수도요금, 폐기물처리비 |
| F | 농림수산품 | 묘목, 잔디, 수목 등 |
| G1~G5 | 표준시장단가 | 토목(G1)/건축(G2)/기계설비(G3)/전기(G4)/통신(G5) |
| Z | 기타비목 | 수수료, 부담금, 보험료, 용역비, 제경비 등 |

> G1~G5 구분 기준: 조달청 물가변동 질의응답집 표준시장단가 세부공종코드 목록

## 입력 엑셀 구조

| 열 | 내용 |
|----|------|
| A열 | 비목군 (출력 대상) |
| B열 | 호표 코드 (`#.숫자` 형식이면 합계행) |
| C열 | 품목 코드 |
| D열 | 품명 (분류 판단의 주요 입력) |
| E열 | 재료비 금액 |
| F열 | 노무비 금액 |
| G열 | 경비 금액 |
| H열 | 합계 금액 |

> **표준시장단가목록 시트** (선택): B열=세부공종코드, C열=공사구분. 없으면 내장 9,607건 코드로만 판별.

## 코드 구조

### 주요 함수

| 함수 | 역할 |
|------|------|
| `process_file(path)` | 전체 처리 진입점. PASS 0.5~4 순서대로 실행 |
| `classify_header(...)` | 호표 행 분류 (내부 참조용, A열 미기재) |
| `classify_sub(...)` | 하위 행 분류 (A열에 기재) |
| `classify_material(name, code)` | 재료비 행 세부 분류 (C/D/E/F 판별) |
| `classify_expense(code, name, ...)` | 경비 행 세부 분류 (B/E/Z/G 판별) |
| `is_std_market(code, name, std_map)` | 표준시장단가 여부 및 G1~G5 결정 |
| `find_std(code, std_map)` | 코드 기반 표준시장단가 매칭 |
| `_scan_sub(ws, row, ...)` | 호표 하위 행에 표준시장단가/견적서 존재 여부 탐색 |
| `_std_nearby(ws, row, ...)` | 주변 5행 내 표준시장단가 유무 확인 |

### 처리 순서 (PASS)

```
PASS 0.5│ GZZZZZZ 분리코드 정규화  ← v8.1 신규
        │ GZZZZZZ 코드 + .M/.L/.E 없는 행을 품명 패턴으로 자동 인식
        │ → 코드에 .M/.L/.E 추가 + C열 노란색 표시 (자동 수정 마커)
        │
PASS 1  │ 호표 행 분류
        │ B열이 #.숫자인 합계행 → 내부 dict(header_bimok)에 저장
        │ A열에는 기재하지 않음 (호표는 비분류 항목)
        │
PASS 2  │ 하위 행 분류
        │ H열 합계가 0이거나 null이면 건너뜀
        │ classify_sub() 호출 → A열에 비목군 기재
        │
PASS 2.5│ 기본비목 미표기
        │ 재료비→D, 노무비→A, 경비→B는 해당 비목의 기본값이므로 A열을 빈칸으로 처리
        │ 예외: 자체 코드가 없고 상위 호표가 G1~G5이면 → G 기재 (소액 경비 제외)
        │
PASS 3  │ 검토 필요 항목 N열 기재
        │ 호표 내 하위행 분류 기준 등 참고 사유 자동 표시
```

### 분류 우선순위 (`classify_sub` 내부)

```
1. E비목 코드·키워드   전력·수도·가스·폐기물 관련 코드/품명 → E
2. 사람 키워드         작업자·기능공·기술자 등 → A
3. M코드 경비          장비 관련 소모품 → D (단, M54W004 물/보충수 → E)
4. 표준시장단가 매칭   _BUILTIN_STD 9,607건 + 엑셀 시트 → G1~G5
5. _코드 / 품명 '표준시장' → G1
6. 기계경비 키워드     크레인·굴삭기·장비 등 → B
7. 주변행 스캔         직전 3행 이내 키워드로 Z 판별
8. 기본비목 반환       재료비→D, 노무비→A, 경비→B
```

### 주변행 스캔 키워드 (Z 분류)

직전 3행 이내의 품명에 아래 키워드가 있으면 Z로 분류합니다.

> 부가가치세, 계측, 부담금, 분담금, 용역비, 작성용역, 제경비, 기술료, 검토비, 문서작성, 교육훈련, 조사비, 평가비, 직접경비, 통행료, 인쇄, 복사, 측정기

### 표준시장단가 내장 목록 (`_BUILTIN_STD`)

조달청 표준시장단가 세부공종코드 9,607건이 딕셔너리 형태로 내장되어 있습니다.
엑셀에 `표준시장단가목록` 시트가 있으면 시트를 우선 사용합니다.

## 출력 파일

입력 파일명에 `_비목분류`가 붙은 새 파일로 저장됩니다.
예) `산근.xlsx` → `산근_비목분류.xlsx`

| 열 | 상태 | 의미 |
|----|------|------|
| A열 | 빈칸 (호표 행) | 합계항목 — 비분류 |
| A열 | 빈칸 (하위 행) | 기본비목(D/A/B)으로 분류됨 — 정상 |
| A열 | 값 있음 | 기본비목이 아닌 다른 비목으로 재분류됨 |
| C열 | 노란색 | GZZZZZZ 코드에 .M/.L/.E 자동 추가됨 (v8.1) |

**A열을 필터링하면 재분류된 항목만 검토 가능**

**N열** (검토사유): 호표 내부 분류 기준 등 참고 정보를 자동으로 기재합니다.

## v8.3 보완 사항 (v8.2 대비)

### 문제 및 수정 내용

#### 1. % /100 계산 분리행 D/A 오표기 수정

**문제**: 품질시험비 3% 분리행 등에서 `재료비: X*3/100 = Y` 형태 행이 D로 잘못 표기됨.

| 행 유형 | v8.2 | v8.3 |
|--------|------|------|
| `재료비: X*3/100 = Y` (코드 없음, G열) | D (오표기) | 미표기 |
| `노무비: X*3/100 = Y` (코드 없음, G열) | A (오표기) | 미표기 |
| `경  비: X*3/100 = Y` (코드 없음, G열) | D (오표기) | 미표기 |
| `재료비(설명): X*Y/100 = Z` | D (정상) | D (유지) |

**근거**: 이들 행은 품질시험비 합계의 3%, 공사비의 X% 등 다른 비목 합계에서 파생된 계산 행으로, 독립적인 재료비/노무비/경비 항목이 아님. 기본비목 미표기 원칙 적용.

#### 2. GZZ .M/.L + % /100 패턴 미표기 수정

**문제**: `GZZZW4010_01.M` 등 공사비 X% 재료비 계산 행이 D로 표기됨.

| 행 유형 | v8.2 | v8.3 |
|--------|------|------|
| `GZZZW4010_01.M` (공사비 1.91% 재료비) | D (오표기) | 미표기 |
| `GZZZW4010_01.L` (공사비 1.91% 노무비) | A (오표기) | 미표기 |
| `GZZ00030A_01.M` (직접 재료비 = 금액) | D (정상) | D (유지) |
| `GZZ00030A_01.L` (직접 노무비 = 금액) | A (정상) | A (유지) |

**판단 기준**: 품명에 `%` + `/100` 패턴이 있으면 파생 계산 행 → 기본비목 미표기. 직접 금액 행은 유지.

#### 3. 외관조사 → Z 키워드 추가

**문제**: `외관조사망도: X*25/100 = Y` 행이 미분류(빈칸)로 처리됨.

**수정**: `외관조사` 키워드를 Z 목록에 추가. 조달청 질의응답집 기준 외관조사는 기타비목(Z).

#### 4. GZZ 경비 라벨 행 + 조사/시험 컨텍스트 → Z

**문제**: `GZZZW21915 경  비: 197,325,000*1 식` 등 1식 조사비 경비 행이 미분류로 처리됨.

**수정**: GZZ 코드 경비 라벨 행에서, 인근 8행 이내에 `추가조사`, `외관조사`, `재하시험`, `구조해석`, `안전성평가`, `조사비`, `내하력평가` 키워드가 있으면 Z로 분류.
