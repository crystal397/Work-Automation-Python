### 2025.03.18 — 하위 행 비목군 분류 추가
- **변경 내용**: 호표 행뿐 아니라 H열(합계)에 값이 있는 모든 하위 행에 비목군 표기
- **분류 원칙**:
  - 표준시장단가(`_`코드 / 품명 '표준시장') 하위 재·노·경 → 상위 G비목 따라감
  - 견적서(GZZ코드) 2개↑ 비목 합산 → 상위 G비목 따라감
  - 견적서 1개 비목만 → D / A / B 개별 분류
  - 일반 공종 하위 → D(재료) / A(노무) / B(경비) 개별 분류
- **경비 라벨 인식 보강**: "경  비:", "-경비", "경   비:" 등 모든 변형 패턴 처리
- **품명 기반 분류**: 시험인→A, 전력요금→E, 장비손료→B, 잡재료비→D

## 알려진 한계

1. **키워드 기반의 근본적 한계**: 등록되지 않은 품명은 오분류 가능하며, 이 경우 프로그램이 스스로 오류를 인지하지 못하여 `?` 표시 없이 잘못 분류될 수 있음
2. **학습 기준 파일 의존**: 남부내륙1공구 산근 파일을 기준으로 규칙과 키워드를 구축하였으므로, 다른 공사 파일에서 새로운 패턴이 나올 경우 초기 검수 후 키워드 보강 필요
3. **규정 해석 영역**: p76 예외 조건(설치비가 재료비의 %인 경우 등)은 사안별 판단이 필요하여 코드로 완전 자동화 불가

## 향후 개선 방향
- 다른 공사 파일 적용 시 발견되는 패턴을 지속 보강
- AI(Claude API) 연동 버전 검토 (품명 의미 이해 기반 분류, 건당 약 0.4원)
