# 비목군 자동분류 핵심 규칙 정리 (v7.0)

> 계약예규, 질의응답집, 이사님 검토1·검토2·메일답변·최신피드백 전체 반영

---

## 1. 비목군이란?

물가변동(계약금액 조정) 시 각 품목의 가격변동률을 어떤 지수로 적용할지 결정하는 분류 기호입니다.

| 기호 | 명칭 | 예시 |
|------|------|------|
| A | 노무비 | 노동자 임금 |
| B | 기계경비 | 크레인, 굴삭기 등 대형 건설기계 |
| C | 광산품 | 잡석, 골재, 모래 등 |
| D | 공산품 | 철근, 시멘트, 페인트 등 공장 생산품 |
| E | 전력·수도·가스·폐기물 | 전기료, 수도요금, 폐기물처리비 |
| F | 농림수산품 | 묘목, 잔디 등 |
| G1~G5 | 표준시장단가 | 토목/건축/기계/전기/정보통신 |
| Z | 기타비목 | 수수료, 부담금, 보험료, 용역비 등 |

---

## 2. 기본 원칙 (계약예규 제68조)

산근에서 각 하위행은 재료비(E열), 노무비(F열), 경비(G열) 중 하나에 금액이 들어가 있습니다. 기본비목은 다음과 같습니다.

- **재료비(E열)만 값 있음 → D (공산품)** — 기본비목
- **노무비(F열)만 값 있음 → A (노무비)** — 기본비목
- **경비(G열)만 값 있음 → B (기계경비)** — 기본비목

**기본비목은 A열에 표기하지 않습니다.** 이사님: *"표시가 되지 않아야 어떤 비목이 유의하게 재분류되었는지 검토/확인하기 위함입니다."*

A열에 값이 있는 것은 **기본비목이 아닌 다른 비목으로 재분류된 것**만 표시됩니다.

---

## 3. 호표 행 처리 (검토2)

**"호표(B열에 #.이 있는 행)는 합계항목이므로 비목군분류 안함"**

- 호표 행의 A열은 항상 빈칸
- 호표의 품명은 하위행 분류 판단에 사용하지 않음
  - 이사님: *"호표행의 문구는 비목군분류시에 고려하지 않고 그냥 하부비목을 집계만 합니다"*
- 단, 프로그램 내부적으로는 호표를 분류하여 하위행의 기준(parent)으로 참조

---

## 4. 표준시장단가(G비목) 판별

판별 우선순위:

1. **코드매칭**: 코드 중간치로 표준시장단가목록(9,607건)과 비교 → 공사구분에 따라 G1~G5 확정
2. **_코드**: 코드가 `_`로 시작하면 → G1 (토목)
3. **품명**: "표준시장" 문구 포함 → G1 (토목)

이사님 메일답변: *"코드매칭 결과대로 분류하고, 코드 외 방법(②③)으로 판단된 경우는 G1로 분류합니다. 산근 시트만 보고는 세부공사 판단이 어렵기 때문."*

제외 대상:
- GZZ 코드 → 표준시장단가 아님
- 짧은 K코드(10자 이하) → 일반공종

---

## 5. E비목 우선 원칙 (메일답변)

**"E는 Z나 G 등 다른 비목군에 우선합니다"**

예: 건설폐기물처리비 하위 반입수수료 (행3335) — '수수료→Z' 규칙과 '폐기물→E' 규칙이 충돌하면, E가 우선하여 E로 분류

---

## 6. 주요 재분류 규칙 (검토1 피드백)

### 경비(B) → 다른 비목으로 재분류

| 항목 | 재분류 |
|------|--------|
| 공구손료, 장비손료(소형), 기구손료 | → D |
| 에어햄머 손료 | → D |
| 1식 운반비/상차비 | → G1 (단, 구역화물→Z) |
| 1식 폐기물처리비 | → E |
| 수수료/제경비/기술료 | → Z |
| 부담금/분담금 | → Z |
| 부가가치세 | → Z |
| 평가비/용역비/시험비/작성비 | → Z |

### 재료비(D) → 다른 비목으로 재분류

| 항목 | 재분류 |
|------|--------|
| 잡석/골재/모래 (K093010 코드) | → C (광산품) |
| 물/식수/수도요금 | → E |
| 묘목/잔디/식재 | → F (농림수산품) |

### 노무비(A) → 다른 비목으로 재분류

| 항목 | 재분류 |
|------|--------|
| 재료비의 OO%인 설치비 | → D |

## 7. 사람 표현 = A (최신피드백)

이사님: *"작업자, 관리인 등 사람으로 표현된 것은 거의 다 A입니다."*

전문작업자, 관리인, 감리원, 기술자, 기능사, 조사원 등 → 경비(G열)에 있어도 **A로 재분류**

---

## 8. 하위행 간 문맥 참조 (최신피드백)

호표 품명은 참조하지 않되, **바로 윗행(하위행 간)의 문맥은 참조 가능**

예: 행3112 "토공비탈면" → 바로 윗행 "1. 계측" → Z로 재분류

스캔 키워드: 계측, 부가가치세, 부담금, 용역비 등

---

## 9. 결과 파일 읽는 법

| A열 상태 | 의미 |
|----------|------|
| 빈칸 (호표 행) | 합계항목 → 비분류 |
| 빈칸 (하위 행) | 기본비목(D/A/B)으로 분류됨 → 정상 |
| 값 있음 | 기본비목이 아닌 다른 비목으로 재분류됨 |

**A열을 필터링하면 재분류된 3,171건만 검토 가능**

N열: 검토사유 (참고용, 111건) — 호표 내부 분류 기준으로 자동 표시
